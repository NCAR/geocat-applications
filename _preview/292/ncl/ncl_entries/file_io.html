Hierarchical Data Format (HDF) and NASA Earth Observing System (EOS) data products

By default, xarray supports HDF5 files, but by specifying the engine, HDF files can be opened as a missing dependency

# HDF and HDF-EOS
import geocat.datafiles as gcd
import xarray as xr

hdf_datafile = xr.open_dataset(gcd.get("hdf_files/avhrr.hdf"), engine="netcdf4")
